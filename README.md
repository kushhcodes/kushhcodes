## 🚀 Featured Projects

### 🎬 Movie Intelligence Engine
> *Content-Based Recommendation System using NLP & ML*

An end-to-end recommendation engine built on the **TMDB 5000 dataset** that delivers personalized movie suggestions using **TF-IDF Vectorization** & **Cosine Similarity**.

- 🔬 Designed a **4-stage NLP pipeline** (tokenization → stop-word removal → stemming → TF-IDF) processing **50,000+ movie descriptions**
- 📊 Conducted EDA across **20+ genres** to identify feature patterns and enhance recommendation quality
- 🖥️ Deployed interactive UI via **Streamlit** for real-time user queries

**Stack:** `Python` `Scikit-learn` `NLP` `TF-IDF` `Streamlit`

### 🔍 PersonaVerify: AI Profile Classifier
> *Fake vs Real Profile Detection using Explainable ML*

A full-stack ML system that classifies social profiles as genuine or fake using trained classification models — with real-time predictions via a REST API.

- 🤖 Trained & evaluated **3 ML models** (Logistic Regression, Decision Tree, Random Forest) — achieving **88–92% accuracy** with Random Forest
- ⚙️ Engineered **8+ features** (engagement ratios, profile signals) to reduce false positives
- 🌐 Built a **Django REST API + React** frontend for live predictions with confidence scoring
- 💡 Used **Random Forest feature importance** to provide explainable AI (XAI) insights

**Stack:** `Python` `Pandas` `Django REST Framework` `Joblib` `React`
